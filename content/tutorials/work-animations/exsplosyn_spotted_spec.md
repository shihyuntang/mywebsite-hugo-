### Description

This animation illustrates the variations in the stellar spectrum as a function of rotational phase. The SPIRou observations are plotted in black, while the best-fit forward modeling sequence from `EXSPLOSYN` is shown in red. For comparison, the dashed blue line represents a two-component composite spectrum constructed using fixed atmospheric parameters ($T_{\rm phot} = 3914$ K, $T_{\rm spot} = 2769$ K, and a cool spot filling factor $f_{\rm cool} = 55\%$) derived by Gully-Santiago et al. (2017).
